## 2. Google Search Console

**URL**: https://search.google.com/search-console

### Steps to Set Up

1. Go to https://search.google.com/search-console
2. Click **"Start now"** and sign in with Google
3. Choose **"URL prefix"** property type
4. Enter: `https://altaiventures.co`
5. Click **Continue**

### Verification Methods (pick one)

**Option A -- HTML Meta Tag** (easiest if you control the code):
1. Copy the meta tag Google provides (looks like `<meta name="google-site-verification" content="..." />`)
2. Add it to the `<head>` section of your site's root layout
3. Click **Verify** in Search Console

**Option B -- DNS Record** (recommended for full domain coverage):
1. Copy the TXT record Google provides
2. Go to your domain registrar (wherever altaiventures.co is registered)
3. Add a DNS TXT record with the provided value
4. Wait 10-30 minutes, then click **Verify**

### After Verification

- [ ] **Submit sitemap**: Go to Sitemaps > Enter `sitemap.xml` > Click Submit
- [ ] **Request indexing**: Go to URL Inspection > Enter `https://altaiventures.co` > Click "Request Indexing"
- [ ] Also request indexing for key pages:
  - `https://altaiventures.co/properties`
  - `https://altaiventures.co/about`
  - Any individual property pages
- [ ] Check back in 3-5 days to review **Coverage** report for any errors
- [ ] Set up **email notifications** for critical issues (Settings > Email preferences)

## 8. Technical SEO Checklist

These items are being implemented in the codebase. Verify each is working:

- [x] **Sitemap** (`/sitemap.xml`) -- auto-generated, submitted to Google Search Console
- [x] **robots.txt** (`/robots.txt`) -- allows all crawlers, references sitemap
- [x] **Structured data** -- JSON-LD for Organization, LocalBusiness, LodgingBusiness
- [x] **Meta tags** -- unique title and description per page
- [x] **OG images** -- Open Graph images for social sharing
- [ ] **Canonical URLs** -- ensure each page has a `<link rel="canonical">` tag
- [ ] **Mobile-friendly** -- test at https://search.google.com/test/mobile-friendly
- [ ] **Page speed** -- test at https://pagespeed.web.dev/ (aim for 90+ on mobile)
- [ ] **SSL certificate** -- confirm HTTPS is working (green padlock)
- [ ] **404 page** -- custom 404 that links back to homepage
- [ ] **Alt text on images** -- descriptive alt text with local keywords
- [ ] **Internal linking** -- every blog post links to property pages and vice versa
- [ ] **Schema markup validation** -- test at https://validator.schema.org/ or https://search.google.com/test/rich-results
- [ ] **Google Analytics** -- set up GA4 at https://analytics.google.com/
- [ ] **Google Tag Manager** (optional) -- https://tagmanager.google.com/
